# swordknights
Inspired from nidhogg and prince of persia i like to build something for fun

## Requirements 

pip install pygame


## Control 

Player 1 (Left)

w = up
a = left
s = down
d = right

Player 2 (right)

up = up
left = left
down = down
right = right
space = attack
